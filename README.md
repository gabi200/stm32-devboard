
# stm32-devboard
Simple, cost-effective **STM32** development board with onboard voltage regulator and I2C EEPROM. The provided files include both the electrical schematic and the PCB layout. The design has been fabricated at JLCPCB and validated in real-life operation. 

**Caution: Please note that the component assembly was performed manually, so the design does not take into consideration automated assembly.**
## Board specifications

 - STM32L4-family microcontroller
 - AMS1117 voltage regulator
 - M24256E-F 256-Kbit I2C EEPROM
 - GPIO and Debug/Programming headers
 - 3.3V I/O voltage
 - Maximum recommended VIN voltage: 12V
